# OWASP-BLT Repository Logos

This directory contains unique red-themed logos for each repository in the OWASP-BLT organization.

## Design Principles

- **Consistent Color Scheme**: All logos use a red gradient (#DC2626 to #991B1B) that aligns with the OWASP-BLT brand identity
- **Unique Identifiers**: Each logo has a distinctive design that reflects the repository's purpose
- **Scalable Format**: SVG format ensures logos look sharp at any size
- **Accessibility**: Logos are designed with sufficient contrast for visibility

## Usage

### Direct Link
Link to individual logos using:
```
docs/assets/img/logos/{repository-name}.svg
```

### In HTML
```html
<img src="./docs/assets/img/logos/BLT.svg" alt="BLT Logo" />
```

### In Markdown
```markdown
![BLT Logo](./docs/assets/img/logos/BLT.svg)
```
